Individual Website Project Site Plan

WDD 231 – Web Frontend Development

Site Name

Explore Uganda

The site name was selected because it clearly represents a website dedicated to showcasing tourist destinations, culture, and travel experiences within Uganda.

Optional domain: explore-uganda.org

Site Purpose

The purpose of this website is to provide visitors with information about popular tourist destinations in Uganda, cultural highlights, travel tips, and recommended activities for both local and international travelers.

Scenarios

  • What are the top tourist attractions to visit in Uganda?
  • Where can I find travel tips and cultural information before visiting Uganda?

Color Scheme

Primary Color: Dark Green (#1b5e20) – used for headings and navigation.

Secondary Color: Light Beige (#f5f5dc) – used for background areas.

Typography

Heading Font: Arial, sans-serif – used for all headings.

Body Font: Verdana, sans-serif – used for paragraph text.

Wireframe

Mobile View

  • Header with logo and navigation
  • Main featured image
  • Content sections stacked vertically
  • Footer with contact and links

Desktop View

  • Header with logo and horizontal navigation
  • Main content displayed in multiple columns
  • Featured sections highlighted
  • Footer across the bottom of the page
